# Employee Management System

A Full Stack Employee Management System built using **Spring Boot**, **React**, and **MySQL**.  
This project allows users to manage employee records with features like adding, updating, deleting, and viewing employee details.

---

## Tech Stack

Frontend
- React
- JavaScript
- CSS

Backend
- Spring Boot
- Java
- REST API

Database
- MySQL

Tools
- Git
- GitHub
- VS Code
- IntelliJ IDEA

---

## Features

- Add new employees
- View employee list
- Update employee details
- Delete employee records
- REST API integration
- Responsive UI

## How to Run the Project

### Backend

Run the Spring Boot application:

mvn spring-boot:run

Backend runs on:

http://localhost:8080

---

### Frontend

Run React application:

npm install  
npm start  

Frontend runs on:

http://localhost:3000
