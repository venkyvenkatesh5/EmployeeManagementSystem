# Employee Management System

A **Full Stack Employee Management System** developed using **Spring Boot, React, and MySQL**.
This application allows users to efficiently manage employee records through a simple and responsive web interface.

---

## 🚀 Tech Stack

### Frontend

* React
* JavaScript
* CSS

### Backend

* Spring Boot
* Java
* REST APIs

### Database

* MySQL

### Tools

* Git
* GitHub
* VS Code
* IntelliJ IDEA

---

## ✨ Features

* Add new employees
* View employee list
* Update employee details
* Delete employee records
* RESTful API integration
* Responsive user interface

## ▶️ How to Run the Project

### Backend

Run the Spring Boot application:

mvn spring-boot:run

Backend will run on:

http://localhost:8080

---

### Frontend

Install dependencies and start React application:

npm install
npm start

Frontend will run on:

http://localhost:3000
